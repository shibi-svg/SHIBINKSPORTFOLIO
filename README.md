# Shibinks Portfolio

A clean, modern single-page portfolio website.

## How to use

- **Open locally**: double-click `index.html` (or right-click and open in your browser).
- **Customize text**: edit the content in `index.html` (name, summary, projects, experience, contact details).
- **Customize styles**: adjust colors, spacing, and layout in `styles.css`.
- **Behavior**: `script.js` handles the mobile navigation, smooth scrolling, theme toggle, and footer year.

## Recommended customizations

- Update:
  - Name, role, and summary in the hero section.
  - Skills in the Skills section.
  - Projects (titles, descriptions, tech stack, and links).
  - Experience (roles, companies, dates, and bullet points).
  - Contact email, location, and social links.
